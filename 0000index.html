3 Preliminary Transformation

3.1 Normalizing Sugar Variables with a min_max ranging function

Normalizing by ranging between min and max for each sugar, specifically \(Y' = \frac{Y - Y_{min}}{Y_{max} - Y_{min}}\). All transformed values will be \(\mathbb{R}\in [0, 1]\). This helps with visual comparisons as all ranges can by portrayed on the same scale but has limited effect on the distribution

The min-max normalizing function is defined:

# Min-Max Normalize Function
# v 0.1
min_max_norm <- function(x) {
  (x - min(x, na.rm = TRUE)) / (max(x, na.rm = TRUE) - min(x, na.rm = TRUE))
}

Apply the min-max normalization transformation to the sugar column values:

# normalize samples table
samples_norm_mm <- samples %>%
  mutate(across(fructose:glycogen, .fns = min_max_norm, .names = "{.col}_norm_mm")) %>%
  select(-fructose:-sample_flush_vol,-year,-month) %>%
  relocate(fructose_norm_mm:glycogen_norm_mm, .before = fine_root_mass)

4.5.1 Linear Model: Glucose + Fructose = Arabitol

Fitting this data to a linear model, we have two possible models:

\[\begin{align} Y &= a_0 + a_1X_1 + a_2X_2 \tag{1},\\\\ Y &= a_0+a_1X_1+a_2X_2+a_{12}X_1X_2 \tag{2} \end{align}\]

Setting the models:

mod1_arab_fruc_gluc <- lm(arabitol_norm_mm ~ fructose_norm_mm + glucose_norm_mm, data = samples_norm_mm)

mod2_arab_fruc_gluc <- lm(arabitol_norm_mm ~ fructose_norm_mm * glucose_norm_mm, data = samples_norm_mm)

Producing a Data Grid for Visualizing the Model Predictions
